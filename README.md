# sse-codec
A [`futures_codec`](https://crates.io/crates/futures_codec) that encodes and decodes Server-Sent Event/Event Sourcing streams.

## Documentation
On [docs.rs](https://docs.rs/sse-codec).

## Installation
In Cargo.toml:
```toml
[dependencies]
sse-codec = "^0.2.0"
```
